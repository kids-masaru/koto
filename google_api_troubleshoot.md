# Google検索エラー（403 Forbidden）の解決ガイド

ログによると、Googleの検索機能を使おうとしましたが、**「APIキーの制限」**によってブロックされています。
これは、お使いのAPIキー（Gemini用）が、Gemini以外の機能を使えないようにロックされているためです。

以下の手順で、キーの制限を緩和してください。

## 手順：APIキーの制限を確認・変更する

1.  **Google Cloud Console** の [認証情報ページ](https://console.cloud.google.com/apis/credentials) にアクセスします。
2.  現在使用している **APIキー**（一覧にある「API key 1」などの名前）をクリックして編集画面を開きます。
3.  **「API の制限 (API restrictions)」** という項目を探します。
    *   もし **「キーを制限しない (Don't restrict key)」** になっていればOKですが、たぶんここは「キーを制限する」になっているはずです。
4.  **「キーを制限する (Restrict key)」** が選ばれている場合：
    *   ドロップダウンリストから、**「Custom Search API」** にチェックを入れて追加してください。
    *   （リストに見つからない場合は、まだAPIを「有効」にしていない可能性があります。その場合は上の検索バーでCustom Search APIを検索して有効化してください）
5.  **「保存 (SAVE)」** をクリックします。

これで、同じ鍵を使って「検索」もできるようになります。
設定変更後、5分ほど待ってからLINEで再度試してみてください。
